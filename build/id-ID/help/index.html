Penyetelan pada 1.1.1.1 membutuhkan dua menit dan tidak memerlukan keahlian teknis atau perangkat lunak khusus. Meskipun Anda pemula di bidang komputer, pilih perangkat Anda di bawah ini untuk panduan penyetelan yang mudah diikuti.

  1. Buka Preferensi Sistem.
  2. Cari Server DNS dan pilih dari menu tarik-turun.
  3. Klik tombol + untuk menambahkan Server DNS dan masukkan 1.1.1.1
  4. Klik + lagi dan masukkan 1.0.0.1 (Ini untuk redundansi.)
  5. Klik + lagi dan masukkan 2606:4700:4700::1111 (Ini untuk redundansi.)
  6. Klik + lagi dan masukkan 2606:4700:4700::1001 (Ini untuk redundansi.)
  7. Klik Oke, lalu klik Apply (Terapkan).
  8. Anda pun siap! Perangkat Anda kini memiliki server DNS yang lebih cepat, lebih privat ✌️✌️
Unduh aplikasi seluler

Meskipun langkah-langkah ini untuk Ubuntu, sebagian besar distribusi Linux mengonfigurasi pengaturan DNS melalui Network Manager (Pengelola Jaringan). Atau, pengaturan DNS Anda dapat disebutkan di /etc/resolv.conf

  1. Klik ikon Aplikasi di bilah menu kiri.
  2. Klik Settings (Pengaturan), lalu Network (Jaringan).
  3. Temukan sambungan Internet Anda di panel kanan, lalu klik ikon gerigi .
  4. Klik tab IPv4 atau IPv6 untuk melihat pengaturan DNS Anda.
  5. Atur toggle “Otomatis” pada entri DNS ke Off (Mati).
    6. Berikan alamat DNS 1.1.1.1 di kolom entri DNS:
    • Untuk IPv4: 1.1.1.1 dan 1.0.0.1
    • Untuk IPv6: 2606:4700:4700::1111,2606:4700:4700::1001
  6. Klik Apply (Terapkan) lalu mulai ulang browser Anda.
  7. Anda pun siap! Perangkat Anda kini memiliki server DNS yang lebih cepat, lebih privat ✌️✌️
  1. Klik di menu Start (Mulai), lalu klik Control Panel (Panel Kontrol).
  2. Klik Network and Internet (Jaringan dan Internet).
  3. Klik Change Adapter Settings (Ubah Pengaturan Adapter).
  4. Klik kanan pada jaringan Wi-Fi yang Anda sambungkan, lalu klik Properties (Properti).
  5. Pilih Internet Protocol Versi 4 (atau Versi 6 jika diinginkan).
  6. Klik Properties (Properti).
  7. Tuliskan setiap entri server DNS yang ada sebagai referensi mendatang.
  8. Klik Use The Following DNS Server Addresses (Gunakan Alamat Server DNS Berikut).
  9. Ganti alamat tersebut dengan alamat DNS 1.1.1.1:
    • Untuk IPv4: 1.1.1.1 dan 1.0.0.1
    • Untuk IPv6: 2606:4700:4700::1111 dan 2606:4700:4700::1001
  10. Klik Oke, lalu Close (Tutup).
  11. Mulai ulang browser Anda.
  12. Anda pun siap! Perangkat Anda kini memiliki server DNS yang lebih cepat, lebih privat ✌️✌️

Konfigurasi router Anda mungkin bervariasi. Baca buku panduan Anda untuk informasi lebih lanjut.

  1. Sambungkan ke jaringan nirkabel pilihan Anda.
  2. Masukkan alamat IP gateaway router Anda di browser.
  3. Jika diminta, isi nama pengguna dan kata sandi Anda. Informasi ini dapat diberi label di router.
  4. Di halaman konfigurasi router, cari pengaturan server DNS.
  5. Tuliskan setiap entri server DNS yang ada sebagai referensi mendatang.
  6. Ganti alamat tersebut dengan alamat DNS 1.1.1.1:
    • Untuk IPv4: 1.1.1.1 dan 1.0.0.1
    • Untuk IPv6: 2606:4700:4700::1111 dan 2606:4700:4700::1001
  7. Simpan pengaturan Anda, lalu mulai ulang browser.
  8. Anda pun siap! Perangkat Anda kini memiliki server DNS yang lebih cepat, lebih privat ✌️✌️
